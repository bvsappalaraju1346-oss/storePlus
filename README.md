## What is this?

A full-cycle data project — 2,500 e-commerce orders cleaned and analysed in Python, then visualised in a fully interactive HTML dashboard with no backend, no framework, no subscription. Just open the file and it works.

Built because small businesses drown in spreadsheet data but can't easily answer basic questions like:
- Which product is actually making money?
- Which customers are worth protecting?
- Which month should I stock up inventory?

This answers all of that — visually, instantly.

---

## What's inside

- **6 tabs** — Overview, Sales Trends, Products, Customers, Insights, Limitations
- **Live search** — type to filter, Tab to confirm, match highlighting
- **Export** — Products, Monthly, Customers, or Full Report as CSV
- **Clickable insights** — each recommendation links to the relevant data
- **Honest limitations tab** — what v1.0 can't do and the roadmap to fix it
